# Vibe Matcher

AI-powered prototype that matches natural-language "vibe" queries to fashion products using semantic embeddings and cosine similarity.

## What's inside
- `src/data.py` — mock product dataset
- `src/embeddings.py` — embedding wrapper (OpenAI optional) + mock semantic embeddings
- `src/engine.py` — VibeMatcher engine (cosine similarity)
- `src/main.py` — CLI demo runner
- `src/tests.py` — edge-case tests
- `notebook/vibe_matcher.ipynb` — Colab-friendly notebook
